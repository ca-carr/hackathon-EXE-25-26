# Hackathon Module

This is the repository for the Blockchains and Quantum Implications short series of lectures.

It is part of the University of Exeter's Hackathon module on the MSc Fintech.

## Topics
- Conceptual blockchains
- The offline blockchain
- Complexities of Blockchain Networking
- Quantum cryptography
- Quantum relevance to blockchain
